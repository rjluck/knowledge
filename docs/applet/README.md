


微信小程序是一种运行在微信内部的轻量级应用程序。

在使用小程序时**不需要下载安装**，用户扫一扫或搜一搜即可打开应用。它也体现了“用完即走”的理念，用户不用关心安装太多应用的问题。它实现了应用“触手可及”的梦想，应用无处不在，随时可用，但又无需安装卸载。


小程序的四大特性：
- 无须安装
- 用完即走
- 无须卸载
- 触手可及
